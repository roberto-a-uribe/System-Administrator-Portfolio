## Overview


This project involved creating and modifying different attributes of AD users and AD groups in home lab using Windows ADUC and Powershell within domain called "BERTO.local".
In this project involved creating and modifying a realistic Active Directory organizational structure within my lab domain (BERTO.Local) using Organizational Units (OUs), user accounts, and security groups by using either ADUC or Powershell comamnds.
